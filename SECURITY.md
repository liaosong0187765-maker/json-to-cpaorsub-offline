# Security Policy

## Sensitive Data

This project is a local conversion toolkit. Input and output files may contain account credentials, OAuth tokens, email addresses, and other private data.

Do not publish real input files, generated output files, tokens, passwords, email lists, or API keys.

If secrets are committed by mistake, rotate or revoke them immediately, then remove them from Git history before publishing the repository.

## Reporting

Please report security issues privately to the repository maintainer instead of opening a public issue with sensitive details.
